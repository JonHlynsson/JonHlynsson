To write a new post, copy this file into `src/posts/`, rename it (the file name becomes the URL), and edit the block at the top:

- `title` — the headline
- `description` — one sentence shown in listings and search results
- `date` — YYYY-MM-DD (controls ordering)
- `postTags` — optional labels shown at the bottom of the post

Then write in plain Markdown below the second `---`.

## Headings look like this

Regular paragraphs need no markup at all. *Italics*, **bold**, [links](https://example.com), and images:

`![Alt text](/assets/my-image.jpg)`

> Blockquotes look like this.

That's the whole workflow: one file per post, push it to GitHub, and the site rebuilds itself.
