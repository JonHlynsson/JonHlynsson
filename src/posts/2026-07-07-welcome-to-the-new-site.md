This site has a new engine. It used to run on WordPress with Elementor; it now runs on [Eleventy](https://www.11ty.dev/), a static site generator, hosted for free on GitHub Pages.

What that means in practice:

- **Faster.** Every page is plain HTML — no database, no plugins, nothing to slow it down.
- **Free.** The only remaining cost is the domain name itself.
- **Simpler to write.** Each blog post is a plain text file. No editor lock-in, no plugin updates, no security patches.

Everything else — the blog, publications, conferences, teaching materials, and CV — lives where it always did.
